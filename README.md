# Sudoku Master

A Python-based Sudoku Solver and Player built using wxPython.

## Features
- Solver mode using backtracking with MRV and bitmasking
- Play mode with difficulty-based puzzles
- Hints, undo/redo, timer, and animation support
- Graphical user interface using wxPython

## How to Run
1. Install dependencies:
   pip install wxPython

2. Run the program:
   python sudoku_solver.py
